# NLP on Mechanic Logs 🚗

## Project Description
This project uses Natural Language Processing (NLP) to simplify and analyze mechanic logs. It takes user input describing vehicle issues and matches it with common problems and solutions from a dataset.

## Features
- Text preprocessing (tokenization, stopword removal)
- Keyword matching
- Problem-solution mapping
- Simple web interface using Flask

## Technologies Used
- Python
- Flask
- NLTK
- Pandas
- HTML, Tailwind CSS

## How to Run
1. Install required libraries
2. Run app.py
3. Open browser and use the interface

## Output
Displays the identified problem and suggested solution based on user input.
